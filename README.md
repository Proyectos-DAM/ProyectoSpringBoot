# Sistema de Suscripciones - Spring Boot

Proyecto de gestión de suscripciones con facturación.

## Ejecutar
1. Abrir en IntelliJ
2. Ejecutar Main.java
3. Ir a http://localhost:8080

## URLs
- /facturacion - Facturas
- /suscripciones - Suscripciones  
- /admin/auditoria - Auditoría
- /h2-console - Base de datos
